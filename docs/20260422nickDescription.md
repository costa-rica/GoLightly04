
# What is Go Lightly?
Go Lightly is a meditation application that allows users to create meditation. The UI is designed to allow users to create lightly guided meditations. Users can insert text that aligns with an affirmation or a thought they want to focus on. Then the create mediation UI allows users to include silences to allow for focus and sprinkle in sounds like the Tibetan bell that will gently bring the user back to focus.

# Why create Go Lightly?
I personally find meditation to be fruitful. I use meditation apps like calm, headspace, and YouTube videos. While many of them provide useful insights and affirmations that I want to include there is too much guidance. I would prefer a little less guidance and more periods of silence with sounds like a Tibetan bell that servers as a marker of time passed, like knowing 90 seconds, 2 minutes, or 5 minutes have passed. But also the Tibetan bell serves as a reminder to come back to the thing you were focusing on whether it be your breath, body, or an affirmation.

# How it works
Go Lightly will allow users to listen to public meditations. When users create meditations they will be public. Only paying users will be able to create private meditations.

The goal will be to have public free meditations permanently but to have a paying tier. 

## Initial pay plan
This is the initial idea for monetizing the application.
### Free tier
The free tier will allow for users to listen to public meditations and create a limited amount of meditations using the website.
### Lifetime subscriptions
There will be 200 lifetime users at $30. After the 200 lifetime subscriptions are used up we will only have free or monthly subscriptions
### Monthly subscription
- $2 a month to create your meditation
- $2 a month for private meditations
- $2 a month for using ai generated meditations
