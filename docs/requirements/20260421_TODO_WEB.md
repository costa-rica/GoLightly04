The existing Next.js app has been renamed to `web_obe/` and is kept on disk as a reference for styling, logos, and UI patterns. A fresh `web/` is being built that uses the same visual design system but is architected around the new API contracts, TanStack tables, and signed-token audio streaming. Once the rebuild is complete and verified, `web_obe/` will be deleted.

**Reuse rule:** Pure-presentation components copy forward unchanged. Anything that touches API shapes, admin tables, or data flow gets rebuilt against `@golightly/shared-types`.
